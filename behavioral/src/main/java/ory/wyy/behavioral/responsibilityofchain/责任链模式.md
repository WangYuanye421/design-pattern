# 责任链模式的理解:
- 执行逻辑是一个流程,流程中涉及多个环节
- 流程抽象成接口A,提供公共的方法,并提供set方法作为设置其他环节的入口(如下个审批人)
- 流程中的每个环节抽象成对象并实现接口A,对象应存在成员属性表示其他环节(通过set方法获取其他环节对象的引用)
